# 📍 Place Intelligence AI Agent (MCP + Gemini)

## 📌 Overview

This project implements a **Place Intelligence AI Agent** using the **Model Context Protocol (MCP)** and **Gemini model**.  
The agent provides **location-based insights and recommendations** by integrating external tools like maps and weather APIs.

It can suggest places, activities, and plans based on user queries and real-world context.

---

## 🚀 Features

* ✅ Built using **Model Context Protocol (MCP)**
* ✅ Uses **Gemini (Google Generative AI)**  
* ✅ Provides **location-based suggestions**
* ✅ Supports **weather-aware recommendations**
* ✅ Uses **tool-based AI reasoning**
* ✅ Modular and extensible design
